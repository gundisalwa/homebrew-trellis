# kodhama/tap

The **kodhama family's** Homebrew tap — one tap, N formulas. Each product in the
family ([trellis](https://github.com/kodhama/trellis), and others as they ship a
release) owns and pushes its own formula here; this repo doesn't build anything
itself.

```sh
brew install kodhama/tap/trellis
```

Swap `trellis` for any other formula name under `Formula/` as the family grows.

## How formulas stay current

Each formula is a derived resource, regenerated by its own product's release
pipeline — never hand-edited here. A product's `auto-release` workflow dispatches
a `new-release` event to this repo, which re-pins that formula's version + sha256.
See `decision-0032` in the [Trellis repo](https://github.com/kodhama/trellis) for
the pattern, and `kodhama-0001-family-delivery` in
[kodhama/kodhama](https://github.com/kodhama/kodhama) for why the tap is shared
across the family rather than per-product.
